Quick install

Follow these steps in order. First launch fetches models automatically (can take 10–30 minutes depending on connection).

Prerequisites (install these first)

# 1) Install Homebrew (macOS package manager)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2) Install Git (if not already installed)
brew install git

# 3) Install Python 3 (if not already installed)
brew install python@3.11

# 4) Install Node.js (required for npm)
brew install node

# 5) Install PortAudio (required for microphone access)
brew install portaudio

Main installation

# 1) Clone the repository
git clone https://github.com/travisgerrard/openscribe.git
cd openscribe

# 2) Install Python dependencies (use pip3 on macOS)
pip3 install -r requirements.txt

# 3) Install Node.js dependencies
npm install

# 4) Start the app (auto-downloads models on first run)
npm start

Prefer manual model download? See advanced steps below.

Common issues & solutions

  • "git: command not found" → Install Git first (step 2 above)
  • "pip: command not found" → Use pip3 instead of pip on macOS
  • "npm: command not found" → Install Node.js first (step 4 above)
  • PyAudio errors → Install PortAudio first (step 5 above)
  • Permission errors → Run Terminal with admin privileges
  • Python path errors → See troubleshooting section below

Manual model setup (optional)

# Create folders
mkdir -p models vosk

# Parakeet speech model (~1.5GB)

System requirements

  • OS: macOS (primary), Windows, or Linux
  • CPU: Modern 4‑core or better. Apple Silicon M1/M2+ or Intel/AMD ≥2.5 GHz recommended.
  • Memory (RAM): Minimum 8 GB. Recommended 16 GB for smooth proofreading models; 24–32 GB if using larger 14B LLMs.
  • Python: 3.8 or newer
  • Node.js: 16 or newer
  • Disk space: 5–15 GB for models (recommend ≥20 GB free)
  • Internet: Required on first run for auto‑downloaded models
  • Microphone: Any input supported by your OS

Notes: the default Parakeet speech model is ~1.5 GB and runs well on 8–16 GB RAM. For optional proofreading LLMs, 8B‑class 4‑bit models are comfortable on 16 GB; 14B‑class 4‑bit may require 24 GB+.

Hotkeys & wake‑words

  • ⌘⇧A — Toggle active
  • ⌘⇧D — Start dictation
  • ⌘⇧S — Stop dictation
  • ⌘⇧P — Proofread selection
  • Say “note” / “dictate” — start dictation
  • Say “proof” — start proofreading

Customize hotkeys and wake‑words in Settings → Shortcuts.

Troubleshooting

  • Models won’t download: Check internet connection & free disk (≥20 GB), then relaunch with admin privileges.
  • Vosk not found: Ensure the extracted folder lives under vosk/ (e.g., vosk/vosk-model-small-en-us-0.15/).
  • Crashes on start: Verify Python & Node versions; run pip3 install -r requirements.txt and npm install again.
  • Microphone not working: Check System Preferences → Security & Privacy → Microphone permissions for Terminal/your terminal app.
  • PyAudio import errors: Ensure PortAudio is installed: brew install portaudio
  • Permission denied errors: Run Terminal with admin privileges or check folder permissions.
  • Python version issues: Use python3 and pip3 commands on macOS.
  • Python path errors (ENOENT): The app is looking for a Python path that doesn't exist. Use export PYTHON_PATH=$(which python3) before npm start.

Python path configuration (if you get "ENOENT" errors)

If you see errors like "spawn /path/to/whisper_env/bin/python ENOENT", the app is looking for a Python path that doesn't exist on your system.

# Option 1: Set environment variable before starting
export PYTHON_PATH=$(which python3)
npm start

# Option 2: Create a startup script
echo 'export PYTHON_PATH=$(which python3)' > start.sh
echo 'npm start' >> start.sh
chmod +x start.sh
./start.sh

# Option 3: Check your Python installation
which python3
python3 --version

The app will now use python3 by default, but you can override it with PYTHON_PATH if needed.
